## Course Description

This course teaches you how to test software without writing code, using practical black box testing techniques applied in real-world software teams.

### What You Will Learn

- Understand software requirements from a user's perspective
- Design effective manual test cases using black box techniques
- Perform exploratory testing to uncover hidden defects
- Test web and mobile applications systematically
- Write clear, professional bug reports that developers can act on
- Collaborate effectively with developers and product managers in Agile teams

### Course Approach

The course focuses on **hands-on practice**, not theory. You will work with real application scenarios and complete a final manual testing project that can be used as part of your QA portfolio.

### Who Is This Course For?

- **Beginners** with no programming background
- **Career switchers** entering the QA / testing field
- **Junior developers** who want to strengthen their testing mindset

### Course Outcome

By the end of the course, you will have the skills and confidence required to perform the role of a **Junior Manual QA / Software Tester** in a professional environment.
